# Libft

## Description

Libft is the first project of the 42 curriculum.
The goal of this project is to recreate a set of standard C library functions, as well as additional utility functions that will be useful for future projects.

## Instructions

Compile the library with:

```bash
make
```

Other available commands:

```bash
make clean
make fclean
make re
```

The compiled library will be generated as:

```bash
libft.a
```
